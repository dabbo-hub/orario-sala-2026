# Orario Sala 2026

This project aims to provide a comprehensive scheduling solution for events in 2026.

## Features
- User-friendly interface
- Event management
- Notification system

## Installation
1. Clone the repository: `git clone https://github.com/dabbo-hub/orario-sala-2026.git`
2. Install dependencies: `npm install`
3. Start the application: `npm start`

## Usage
After starting the application, you can access it through your browser at `http://localhost:3000`.
